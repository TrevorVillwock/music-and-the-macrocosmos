Conway's Game of Sound

Conway's Game of Life is a type of cellular automaton invented by the British mathematician John Conway in 1970.

Cellular automata are mathematical systems defined by a set of cells that interact with the cells around them according to a set of rules. The Game of Life is a 2D automaton, but cellular automata with any number of dimensions are possible.

Each cell has its own state defined by a certain number of parameters; in Conway's Game of Life the cells have only one parameter, which is whether they are "alive."

The rules according to conwaylife.com are as follows:

  • Any live cell with fewer than two live neighbours dies (referred to as underpopulation).
  • Any live cell with more than three live neighbours dies (referred to as overpopulation).
  • Any live cell with two or three live neighbours lives, unchanged, to the next generation.
  • Any dead cell with exactly three live neighbours comes to life.

    • Conventionally, the game of life takes place on an infinite grid, but this version uses a finite 10x10 one. This creates interesting effects along the borders as cells with only 3 or 4 neighbors will respond differently to the rules.
